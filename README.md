# RGM Product Segmentation

Segments products by sales behavior and promotional responsiveness for
differentiated RGM strategy recommendations.

## Setup
```bash
pip install -r requirements.txt
```

## Structure
- `src/` — feature engineering, clustering, interaction, and anomaly-detection modules (with tests)
- `notebooks/` — run in order from the file
- `data/raw/` — input data (not tracked here)

## Run
Open notebooks in VS Code / Jupyter and run top to bottom in the order above.
